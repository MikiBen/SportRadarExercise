#Match Analyzer

# Aplikacja służy do:

- odczytaniu danych z pliku "BE_data.json" zawierający opisy meczów
- posortowaniu meczy pod względem najbardziej prawdobodonych wyników
- wpisaniu przez użytkownika ilości wyświetlanych wyników
- stworzenie listy wszystkich zespołów
- wyświetlenie nazw zespołów w porządku alfabetycznym

# Kompilacja
- jakl urochomic z cmd java w wersji 11
